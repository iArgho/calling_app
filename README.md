# calling_app

A Flutter project for real-time **video calling** using the **Agora SDK**. This app allows users to make one-to-one video calls with local and remote video streams, handling permissions and responsive UI.

---

## Features

- Real-time video calling
- Local and remote video views
- Dynamic permission handling for camera and microphone
- Responsive design with `flutter_screenutil`
- Clean architecture with separated Agora service

---

## Getting Started

This project is a starting point for a Flutter application. Follow these steps to run it:

### Prerequisites

- Flutter SDK >= 3.10
- Dart >= 3.10
- An Agora account for **App ID** and **Temporary Token**
- Android/iOS device or emulator with camera and microphone
